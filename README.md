# Manipulacao-de-Datas-e-Calendario
Exercícios de AEDS I (UNIMONTES) sobre funções. Implementação de algoritmos de calendário para verificar anos bissextos, realizar conversão bidirecional entre datas Gregorianas e Julianas, e calcular o dia da semana. O projeto foca na modularização do código e aplicação de fórmulas matemáticas complexas em C.
